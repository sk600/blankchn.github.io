数据类型总结

java是强类型语言，基本数据类型一共有8种

  • 整型——4种
  • 浮点型——2种
  • 字符类型——1种
  • 真值——1种

    整数

    4种整数

  • int 4字节 -20亿～20亿
  • short 2字节
  • byte 1字节 -128～127
  • long 8字节 使用L做后缀（尽量不要使用l，因为与1很难区分）
    注意：java没有任何无符号的整型

    进制表示

  • 16进制使用0x或0X做前缀。eg：0xCAFE。
  • 8进制使用0做前缀（8进制容易混淆，尽量不要使用8进制常数）。eg：010——8。
  • 2进制使用0b或0B做前缀（java 7开始）。eg：0b1001——9。
  • java7开始，可以为数字字面量加下划线（这些下划线只是为了让人易读，编译器会自动去除这些下划线）。eg：1_00_000。

    浮点数

    2种浮点数

  • float 4字节 后缀必须加F或f。
  • double 8字节 后缀可以加D或d，也可以不加。
    16进制可以表示浮点数，0.125=2^(-3)可以表示成0x1.0p-3，p表示指数而不是e。尾数采用16进制，指数采用10进制。指数的技术是2，而不是10
